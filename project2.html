Project 02 — Website Individu

Web Biodata Pribadi

Website profil dan biodata diri menggunakan HTML & CSS

Jenis Project Individu
Materi HTML & CSS
Tujuan Menampilkan biodata diri
Status Selesai ✓

Project ini adalah website biodata pribadi sederhana yang saya buat secara individu. Website ini menampilkan informasi diri seperti nama, kelas, sekolah, hobi, cita-cita, dan foto profil dalam tampilan yang rapi dan terstruktur.

Melalui project ini, saya belajar bagaimana menyusun struktur HTML yang baik dan mendesain tampilannya menggunakan CSS agar terlihat menarik dan mudah dibaca. Project ini menjadi fondasi penting sebelum saya melanjutkan ke materi JavaScript.

Teknologi yang Digunakan

HTML5 CSS3 Flexbox Responsive Design
