Project 01 — Mini Project Kelompok

Emirates Stadium

Website Informasi Stadion Ikonik Arsenal FC

Jenis Mini Project Kelompok
Materi HTML, CSS, JavaScript
Tema Emirates Stadium — Arsenal FC
Status Selesai ✓

Project ini merupakan mini project kelompok yang bertujuan membuat website informatif tentang Emirates Stadium, stadion kebanggaan Arsenal FC yang berlokasi di Holloway, London. Website ini menampilkan berbagai informasi seputar stadion, mulai dari sejarah, kapasitas, hingga fakta menarik.

Dalam project ini, kelompok kami membagi tugas secara merata, mulai dari pembuatan struktur HTML, desain tampilan menggunakan CSS, hingga penambahan interaksi sederhana menggunakan JavaScript. Hasilnya adalah sebuah website yang informatif dan menarik secara visual.

Teknologi yang Digunakan

HTML5 CSS3 JavaScript Responsive Design Flexbox
